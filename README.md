# Motores de Desarrollo 1 🎮

Repositorio correspondiente a la materia **Motores de Desarrollo 1**, donde se almacenan los trabajos prácticos, ejercicios y proyectos realizados durante la cursada.

El objetivo de la materia es introducir los conceptos fundamentales del desarrollo de videojuegos y aplicaciones interactivas utilizando motores de desarrollo modernos.

---

## 📚 Contenidos de la materia

A lo largo del repositorio se pueden encontrar ejemplos y prácticas relacionadas con:

- Introducción a los motores de desarrollo
- Conceptos básicos de videojuegos
- Manejo de escenas y assets
- GameObjects y componentes
- Física y colisiones
- Scripting
- Interacción con el jugador
- Lógica de juego
- Prototipos y demos

---

## 🛠️ Tecnologías utilizadas

- **Motor de desarrollo**: Unity
- **Lenguaje de programación**: C#
- **Control de versiones**: Git / GitHub
